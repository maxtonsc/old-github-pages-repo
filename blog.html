The start of the App project

March 12, 2017 6:13 PM


The concept for this project is essentially to help bridge the communication gap between the general populus of New Zealand and its political representatives. Currently the process is slow on desktop, and not optimized at all for mobile users. Thus, my multiplatform orientated project would be addressing this issue.
The current tech route that I am investigating is a MEAN stack (MongoDB, Express.js, Angular.js, Node.js). Furthermore, to bring it to mobile devices I'm thinking of using Iconic.js which is built around angular and should allow me to create at the very least a strong prototype for both iOS and Android very quickly. Time is an important issue in this project as 2017 is election year and politics is even moreso on people's minds then normal.

Students point of view of the current state of online web dev resources

March 3, 2017 2:15 PM

Trying to find good resources to practice web development is surprisingly limited online. That is not to say that they are not many resources out there, in fact the problem is an oversaturation of poor material. For instance as of writing this post I have tried Udacity, Pluralsight, freecodecamp, and around 5 small sites through reddit forums etc. Instead of continuing a fruitless search I have instead opted to just set out doing projects. Since I have a strong foundation of clean code which I learnt from the University of Washington I figure this approach hopefully wont lead to some ugly code that you see from other self learnt developers.
